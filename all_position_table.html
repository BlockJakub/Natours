CSS Positioning Guide

Position Meaning Example
static Default positioning (normal document flow) div { position: static; }
relative Positions relative to its original location div { position: relative; top: 20px; }
fixed Positions relative to the viewport (stays in place) header { position: fixed; top: 0; }
absolute Positions relative to its nearest positioned ancestor .box { position: absolute; top: 50px; left: 100px; }
sticky Toggle between relative and fixed based on scroll header { position: sticky; top: 0; }
inherit Inherits position from its parent element .child { position: inherit; }
initial Resets position to default (static) .element { position: initial; }

Conclusion

  • static → Default position, follows normal document flow.
  • relative → Positioned relative to its normal position.
  • fixed → Positioned relative to the viewport, stays in place when scrolling.
  • absolute → Positioned relative to its nearest positioned ancestor.
  • sticky → Toggle between relative and fixed based on scrolling.
  • inherit → Inherits positioning from the parent.
  • initial → Resets to default (static).
